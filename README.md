2. Abstract


The Movie Ticket Booking System is a web-based application designed to simulate an online movie ticket reservation platform. It allows users to browse available movies, select show timings, choose seats interactively, and perform mock payments. The system demonstrates Object-Oriented Analysis and Design (OOAD) principles such as modular design, user interaction flow, and system organization. The project focuses on providing a simple and intuitive user interface while implementing core functionalities of a real-world booking system.







3. Introduction



With the increasing demand for online services, movie ticket booking systems have become essential in modern entertainment platforms. This project aims to replicate the basic functionalities of such systems using web technologies. It provides users with a seamless experience for selecting movies, booking seats, and managing bookings.






4. Problem Statement



Traditional ticket booking methods are time-consuming and inconvenient. There is a need for a system that:



Allows users to book tickets online

Displays real-time seat availability

Provides an interactive and user-friendly interface




5. Objectives



To design an interactive movie booking interface

 To implement seat selection with real-time updates

 To simulate a payment system

 To maintain booking history for users







6. System Architecture



The system follows a client-side architecture:



Frontend: HTML, CSS, JavaScript

Logic Handling: JavaScript functions manage user interactions

Data Storage: Browser-based (local storage / session handling)



 7. Technologies Used


HTML: Structure of the web pages

CSS: Styling and layout design

JavaScript: Dynamic behavior and logic implementation





8. Functional Requirements



Display list of movies

Allow selection of show timings

Enable seat selection

Show real-time seat availability

Process mock payments (Card/UPI simulation)

Display booking confirmation

Maintain booking history



9. Implementation Details



The system is divided into multiple modules:


Movie Selection Module: Displays available movies

Seat Booking Module: Interactive grid for selecting seats

Payment Module: Simulates payment process

Booking History Module: Stores and displays previous bookings



JavaScript handles all dynamic interactions and updates the UI accordingly.







 10. Results / Output



The system successfully:



Displays movies and show timings

Allows users to select seats interactively

Simulates a payment process

Generates booking confirmation
 \
Stores booking history
